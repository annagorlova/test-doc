LastOuterDocflow
================

.. code-block:: protobuf
 
  message LastOuterDocflow {
      required string ParentEntityId = 1;
      required OuterDocflowInfo OuterDocflow = 2;
  }

Структура содержит информацию о последнем состоянии внешнего документооборота, например, последний статус, полученный от ГИС МТ "Честный ЗНАК" по обработке документов с маркированными товарами.

-  *ParentEntityId* - идентификатор сущности, по которой получено последнее состояние в рамках внешнего документооборота. Это может быть документ либо запрос на аннулирование.
-  :doc:`OuterDocflow <OuterDocflowInfo>` - содержит подробную информацию о внешнем документообороте.
