Routing
=======

.. code-block:: protobuf

    message Routing
    {
        required bool Kpp = 1;
        required bool Address = 2;
    }

Структура содержит информацию о доступном для подразделения способе маршрутизации.

- *Kpp* - доступна ли маршрутизация по КПП. Если поле имеет значение true, то КПП подразделения должен быть заполнен.
- *Address* - доступна ли маршрутизация по адресу. Если поле имеет значение true, то адрес подразделения должен быть заполнен.
