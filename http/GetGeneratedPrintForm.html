GetGeneratedPrintForm¶

Имя ресурса: /GetGeneratedPrintForm

HTTP метод: GET

Параметры строки запроса:

  • printFormId - идентификатор печатной формы, полученный с помощью метода GeneratePrintFormFromAttachment;

В запросе должен присутствовать HTTP-заголовок Authorization с необходимыми данными для авторизации.

Если печатная форма еще не построена, ответ будет содержать http-заголовок Retry-After с примерной оценкой времени (в секундах), оставшегося до завершения.

Клиент может повторить вызов GetGeneratedPrintForm через указанное время.
